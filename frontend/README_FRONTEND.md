# 🎨 Frontend: Interfaz del Descargador de Playlists

Este es el frontend del proyecto, construido con **Astro**, **React** y **Tailwind CSS**. Aquí crearemos una interfaz sencilla y bonita donde el usuario podrá pegar el enlace de Spotify y ver el progreso de descarga.

## 🛠️ Requisitos Previos

- Tener instalado **Node.js** en tu ordenador (puedes comprobarlo escribiendo `node -v` en la terminal).

## 🚀 Paso a Paso: Creación del Proyecto

### Fase 1: Iniciar el proyecto de Astro

1. Abre tu terminal en la carpeta principal de tu proyecto.
2. Ejecuta el siguiente comando para crear el proyecto base:
   ```bash
   npm create astro@latest frontend
   ```
